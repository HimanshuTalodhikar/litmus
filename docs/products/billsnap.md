# BillSnap

## Overview
BillSnap is an automated invoicing and billing platform for SMBs. It handles recurring billing, payment collection, and financial reporting.

## Features
- Recurring billing with flexible subscription plans
- Automated payment reminders before due dates
- Multiple payment methods: credit card, ACH, wire transfer
- Customizable invoice templates with your branding
- Tax calculation and compliance for US and EU
- Real-time payment tracking and reconciliation
- API access for custom integrations

## Pricing
- **Starter**: $29/month — up to 50 invoices/month
- **Growth**: $79/month — unlimited invoices, API access
- **Enterprise**: $199/month — white-label, dedicated account manager

## Status
Launched in Q2 2025. Current version: 1.8.0

## Integrations
Stripe, PayPal, QuickBooks, Xero, Slack, Zapier, API
