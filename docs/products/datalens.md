# DataLens

## Overview
DataLens is a product analytics platform that helps teams understand how users behave. Track events, build funnels, and measure retention — without writing SQL.

## Features
- Event tracking with no-code and code-based SDKs
- Funnel analysis for conversion optimization
- Cohort analysis and retention charts
- User segmentation with custom properties
- Real-time dashboards and alerts
- A/B testing with statistical significance
- Session recording and heatmaps

## Pricing
- **Free**: up to 10,000 monthly active users
- **Pro**: $49/month — up to 100,000 MAU, unlimited dashboards
- **Business**: $199/month — 1M MAU, session recording, A/B testing

## Status
Beta launched in Q3 2025. Current version: 0.9.2

Segment, Amplitude, Mixpanel, Zapier, Slack, API
